# The-Heist-Escape-Loop
Got you — more natural, less robotic 👇  **The Heist &amp; Escape Loop** is a 2D C++ (Raylib) game that makes DAA concepts feel real through gameplay. Each level is based on algorithms like Knapsack, BFS, Sorting, and TSP, where your choices actually change difficulty, guard behavior, and results—so you learn by playing, not just studying.
