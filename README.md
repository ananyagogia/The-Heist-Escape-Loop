# The Heist & Escape Loop
# Project Description

This is an interactive game built using C++ and Raylib to demonstrate core concepts of Design and Analysis of Algorithms (DAA).
Each level represents a different problem where the player chooses an algorithm, and the game shows how that choice affects performance and outcome.

# What We Used
Language: C++
Library: Raylib
Algorithms:
Knapsack (Greedy vs DP)
BFS & DFS (Maze + Pathfinding)
Sorting (Bubble vs Quick Sort)
TSP (Brute Force vs DP)
# How It Works
Game is divided into multiple levels
Each level is based on a real algorithm
Player selects an approach
Score and gameplay change based on algorithm efficiency
# How to Run
1. Install Raylib
2. Compile
g++ main.cpp -o game -lraylib -lopengl32 -lgdi32 -lwinmm
3. Run
./game
